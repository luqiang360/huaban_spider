## Huaban Spider

花瓣网爬虫

## Python2.7.x

#### 依赖库
所有依赖库推荐使用anconda管理安装

* scrapy    scrapy爬虫框架
* pymysql   mysql数据库连接
* DBUtils   mysql数据库连接池
* enum      枚举

#### 使用帮助
* ./main.py -h (Linux)
* python main.py -h (Windows)

## 说明

需先自行手动创建数据库huaban2
